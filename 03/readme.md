#  «Qt Designer»

### Цель задания

В результате выполнения этого задания вы научитесь создавать простую форму в Qt Designer.

------


### Задание

Создайте форму, которая включает:
* 2 кнопки;
* 2 текстовых лейбла;
* 2 изменяемых виджета (например, LineEdit, SpinBox);
* 2 различных стиля компоновки.
